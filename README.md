# ---
Сайт-портфолио с функциями отправки данных на почту(Node.js), копирование данных почты и телефоны в буфер-обмена(VanilaJS), анимация заполнения(VanilaJS), анимация при прокрутке страницы(Animate.css + wow.js)
Как добавить проект к себе на компьютер:
1) Скачайте git с официального сайта;
2) Создайте пустую папку, откройте консоль - в ней вам предстоит писать команды;
3) git init - инициализирует отслеживание git в вашей папке;
4) git remote add https://github.com/wouldRewind/MyOwnSite.git master
5) git pull master
